# **World Economic News - Project Blueprint**

## **프로젝트 개요**
매일 전 세계 주요 경제 뉴스 5가지를 선정하여 제공하는 뉴스 큐레이션 사이트입니다. 사용자는 원하는 언어와 날짜를 선택하며, 다크/화이트 모드 테마를 자유롭게 전환할 수 있습니다.

## **디자인 및 스타일 가이드**
- **Aesthetics:** `oklch` 컬러 시스템을 활용한 선명한 색감.
- **Themes:** 다크 모드(기본) 및 화이트 모드 지원.
- **Typography:** 'Inter' 시스템 폰트 및 가변 굵기 강조.
- **Interactivity:** 테마 토글, 뉴스 펼치기 애니메이션, 날짜 필터링.

## **구현 계획**

### **1단계: 구조 및 스타일 정의 (완료)**
- 기본 레이아웃 및 현대적인 CSS 설계.

### **2단계: Web Components 및 로직 구현 (완료)**
- `<news-card>` 컴포넌트 및 다국어 엔진 구축.

### **3단계: 기능 고도화 (완료)**
- 날짜 선택 및 상세 내용 펼치기 기능 추가.

### **4단계: 테마 시스템 도입 (진행 중)**
- 다크/화이트 모드 토글 버튼 및 CSS 변수 체계 구축.
- 테마 설정 로컬 스토리지 저장 기능.
