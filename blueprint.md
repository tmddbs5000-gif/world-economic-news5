# **World Economic News - Project Blueprint**

## **프로젝트 개요**
매일 전 세계 주요 경제 뉴스 5가지를 선정하여 제공하는 뉴스 큐레이션 사이트입니다. 사용자는 원하는 언어와 날짜를 선택하여 각 뉴스의 상세 내용(약 500자)을 확인할 수 있습니다.

## **디자인 및 스타일 가이드**
- **Aesthetics:** `oklch` 컬러 시스템을 활용한 선명한 색감과 미세한 노이즈 텍스처 배경.
- **Typography:** 가독성이 뛰어난 시스템 폰트 스택 및 중요 키워드 강조.
- **Interactivity:** 뉴스 카드 호버 효과, 부드러운 언어 전환 애니메이션, **상세 내용 펼치기/접기**.
- **Components:** Web Components 기반의 독립적이고 선언적인 UI 요소.

## **구현 계획**

### **1단계: 구조 및 스타일 정의 (완료)**
- `index.html`: 언어 선택기(Selector)와 뉴스 리스트 컨테이너 레이아웃 구성.
- `style.css`: 현대적인 CSS를 활용한 스타일링.

### **2단계: Web Components 및 로직 구현 (완료)**
- `NewsCard` 컴포넌트 개발: 제목, 요약, 이미지, 날짜 등을 포함하는 커스텀 엘리먼트.
- `main.js`: 다국어 데이터 처리 로직 및 UI 업데이트 엔진 구축.

### **3단계: 기능 고도화 (진행 중)**
- **날짜 선택 기능:** 헤더에 날짜 선택기(Date Picker) 추가 및 날짜별 데이터 필터링.
- **상세 내용 펼치기:** 500자 내외의 긴 요약문을 '더 보기' 버튼으로 제어하는 기능 추가.
- **데이터 보강:** 2026.04.13일 및 과거(04.12일) 데이터 추가.
