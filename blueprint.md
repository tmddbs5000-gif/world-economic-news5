# **World Economic News - Project Blueprint**

## **프로젝트 개요**
매일 전 세계 주요 경제 뉴스 5가지를 선정하여 제공하는 뉴스 큐레이션 사이트입니다. 다국어 지원, 테마 전환, 그리고 외부 제휴 문의 접수 기능을 제공합니다.

## **디자인 및 스타일 가이드**
- **Aesthetics:** `oklch` 기반의 프리미엄 다크/라이트 테마.
- **Interactivity:** 테마 토글, 뉴스 펼치기, 제휴 문의 모달.
- **Forms:** Formspree를 활용한 서버리스 문의 접수 시스템.

## **구현 계획**

### **1-3단계: 핵심 기능 (완료)**
- 뉴스 큐레이션, 다국어 지원, 날짜 선택, 테마 전환 구현 완료.

### **4단계: 제휴 문의 기능 도입 (완료)**
- Formspree 연동 제휴 문의 폼 제작.
- 세련된 애니메이션이 적용된 모달(Modal) UI 구현.
- 폼 유효성 검사 및 사용자 피드백(성공 메시지) 로직 추가.

### **5단계: 동적 뉴스 업데이트 시스템 (완료)**
- 한국 시간(KST) 기준 오전 10시를 기점으로 매일 새로운 뉴스가 업데이트되도록 구현.
- 오전 10시 이전에는 전일 뉴스, 10시 이후에는 당일 뉴스를 기본으로 노출.
- 하드코딩된 날짜 대신 `new Date()`와 타임존 오프셋을 활용한 동적 날짜 계산 로직 도입.
- 뉴스 데이터가 없는 날짜에 대해서도 세련된 플레이스홀더 뉴스를 생성하여 끊김 없는 서비스 제공.
